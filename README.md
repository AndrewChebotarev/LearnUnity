# 📚 Изучение Unity

Добро пожаловать в репозиторий, посвящённый моему пути в освоении игрового движка **Unity**.  
Здесь собраны проекты, учебные материалы, заметки и наработки, созданные в процессе обучения.

## 🎯 Цель репозитория

- Систематизировать знания по Unity.
- Закреплять теорию на практике через небольшие игровые проекты.
- Документировать интересные решения, полезные скрипты и типичные ошибки.
- Создать портфолио работ для дальнейшего профессионального роста.
